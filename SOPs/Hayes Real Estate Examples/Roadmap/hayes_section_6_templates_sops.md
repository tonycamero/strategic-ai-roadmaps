# 6.2 WORKFLOW LIBRARY
## A. Lead Intake Workflow (GHL)
1. Trigger: New lead enters via any channel.
2. Instant SMS + email sent.
3. Qualification form or SMS questions.
4. Auto-tag: Buyer/Seller/Referral.
5. Lead scoring applied.
6. Routing: Assign to agent.
7. Task created: “Contact {{lead_name}} immediately.”
8. Follow-up workflow triggered.

---

## B. Follow-Up Workflow
1. Day 1: SMS + Email.
2. Day 3: SMS.
3. Day 5: Email.
4. Day 10: SMS.
5. Day 17: Voicemail drop.
6. Day 30: Re-engagement workflow.
7. Stop/pause if lead responds.

---

## C. Buyer Onboarding Workflow
1. Trigger: Lead converted → Buyer pipeline.
2. Send Buyer Packet (Jotform).
3. Create TC tasks.
4. Create client portal board.
5. Auto-send next steps email.

---

## D. Seller Onboarding Workflow
1. Trigger: Lead converted → Seller pipeline.
2. Send Seller Disclosure Packet.
3. Create TC tasks.
4. Create client portal.
5. Auto-send "Prepare Your Home" checklist.

---

## E. Transaction Coordinator Workflow
1. Trigger: Offer accepted.
2. Create portal board.
3. Auto-send document requests.
4. Create inspection tasks.
5. Create appraisal tasks.
6. Weekly status updates automated.

# 6.3 SOP LIBRARY
## SOP: Lead Intake Handling
**Purpose:** Ensure every lead receives a consistent response.

**Steps:**
1. Open GHL conversations.
2. Respond personally within 2–5 minutes.
3. Update lead status.
4. Assign tasks.
5. Log notes.

---

## SOP: Daily Agent Responsibilities
1. Clear all GHL tasks.
2. Review pipeline.
3. Respond to new leads.
4. Update deal statuses.
5. Check client portal for updates.

---

## SOP: Escalation Handling
1. Check GHL lead timeline.
2. Review client portal status.
3. Document the issue.
4. Notify TC or Ops.
5. Notify Roberta only if needed.

---

## SOP: TC Daily Workflow
1. Review new deals.
2. Monitor document uploads.
3. Update portal.
4. Send reminders.
5. Coordinate inspections.
